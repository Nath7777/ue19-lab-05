# Ue19-lab-05

## Description
Cette application python interroge une API publique (https://blague-api.vercel.app/) pour afficher une blague aléatoire.
Elle permet d'apprendre comment envoyer une requête HTTP avec la librairie 'requests', traiter la réponse JSON, et afficher le résultat.
Ce projet a été réaliser pour le laboratoire : UE19-Lab-05

## Comment installer et exécuter le projet

$ git clone https://github.com/Nath7777/ue19-lab-05.git

$ cd ue19-lab-05

$ pip install -r requirements.txt

$ python app.py
